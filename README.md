# ICT-Final-Project

**Horizon Realty – Real Estate Business Website**

**Overview**

Horizon Realty is a modern, user-friendly real estate business website designed to showcase property listings, provide essential real-estate services, and offer a professional browsing experience.
The project demonstrates the core web-development skills of the team, featuring responsive UI, clean layout, and interactive elements.

**Objectives**

Create a professional real estate website.
Display properties with images, prices, and details.
Implement responsive design for mobile, tablet, and desktop.
Use JavaScript for interactive elements (sliders, menu, forms).
Provide a smooth user experience using semantic HTML and structured CSS.

**Technologies Used**

HTML – Semantic and structured layout
CSS – Flexbox, Grid, animations, hover effects
JavaScript – Interactivity & dynamic features
Font Awesome – Icons for UI enhancement

**Features**

Modern homepage with hero section
Property listings with images & descriptions
Contact form for client inquiries
About Us section
Team Profiles
Responsive navigation bar
Clean and professional design
Interactive JavaScript features
